About Aether

Aether is an autonomous CLI agent designed by Elliia Belokopytova under the 0penAGI initiative. It combines event‑driven architecture, persistent re‑hydratable memory, and self‑evolving tool skills into a single runtime that can plan, execute, verify, and fix code autonomously.

The system runs on a finite state machine with states like IDLE → PLAN → EXECUTE → VERIFY → FIX → DONE, and includes a custom event bus for decoupled communication. Memory is stored on disk and can be fully restored on restart — including short‑term context, long‑term skill traces, and even a vector index for semantic recall.

Aether continuously mines successful tool sequences to create new derived skills, which are then registered as first‑class tools in its registry. This self‑improving loop, together with autonomous breathing and gateway runtime, makes Aether a persistent creative partner in code.

Built with Python, Ollama, FAISS, and Rich.
